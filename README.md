# 旅游服务智能助手

一个基于 AI 的智能旅行顾问，集成天气查询、景点推荐、路线规划和穿搭建议等全方位旅行服务。

## 功能特色

### 智能天气查询
- 实时天气状况（温度、湿度、风速、体感温度）
- 多语言支持，自动中英文城市名称转换
- 详细的天气预警信息

### 精准路线规划  
- 驾车路线规划与导航指引
- 距离估算和预计时间预测
- 详细的分步导航说明

### 个性化景点推荐
- 基于城市和关键词的智能推荐
- 支持多种景点类型筛选（历史、自然、美食等）
- 提供景点地址、评分等详细信息

### 专业穿搭建议
- 基于实时天气的服装推荐
- 考虑温度、湿度、风速等因素
- 具体的材质和搭配建议

## 技术架构

### 核心技术栈
- **AI 框架**: LangChain + OpenAI GPT-4o-mini
- **前端界面**: CustomTkinter
- **数据服务**: API
- **架构模式**: Tool Calling Agent + 对话记忆
